20221006 김나연

## 작품명
우리 식물은 이제 안 아파요

## 작품 설명
식물 주변에 배치하여 온습도를 감지하고 소리와 빛으로 인지 후 MQTT를 이용하여 현재의 상태를 알 수 있게 한다.

## 작품 제작에 사용한 센서와 액추에이터, 역할

작품에서 사용한 센서와 액추에이터는 다음과 같습니다.
- 부저 - 정상범위가 아닐 때 소리로 알려준다.
- LED(노랑, 빨강, 초록) - 식물의 상태를 색으로 나타낸다.
- 온습도센서 - 온도와 습도를 감지한다.
